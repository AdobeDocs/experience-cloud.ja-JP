# web アプリケーション {#web-applications}

Web アプリケーションは、**機能API V3**&#x200B;を呼び出して機能フラグを取得し、その背後にある条件付きロジックをアプリケーションコードに配置することで、エクスペリエンスロールアウトと統合します。

リクエストパラメーター、レスポンス形式、呼び出しの例など、完全なAPI リファレンスについては、このガイドの機能API セクションの&#x200B;**GET機能API V3**&#x200B;を参照してください。

## 統合ステップ {#integration-steps}

統合プロセスの詳細については、[統合手順](integration-steps.md)を参照してください。
