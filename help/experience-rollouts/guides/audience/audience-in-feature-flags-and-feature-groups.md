# 機能フラグと機能グループのオーディエンス {#audience-overview}

エクスペリエンスのロールアウトでは、機能フラグと機能グループに対するオーディエンスターゲティングを提供します。 コンテキスト変数を組み合わせて、どのユーザーが機能を受け取るかを正確に制御できます。

コンテキストベースのターゲティング（例えば、ユーザーのアクティブな言語またはデバイスタイプによるターゲティング）については、[&#x200B; オーディエンスルールでコンテキストを使用](using-context-in-audience-rules.md)を参照してください。
