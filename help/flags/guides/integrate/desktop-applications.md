# デスクトップアプリケーション {#desktop-applications}

デスクトップアプリケーションは、SDKを介してFlagsと統合されます。 デスクトップスタック固有の統合ガイダンスについては、Adobeの担当者にお問い合わせください。

>[!NOTE]
>
>Desktop SDKのドキュメントは現在準備中であり、近日中に公開される予定です。

## アプリケーション ID {#app-id}

デスクトップクライアントは、**製品コードと製品バージョン**&#x200B;をアプリケーション IDとして使用できます。
